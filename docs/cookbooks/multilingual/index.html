Setup a multilingual bot

  1. Add languages under the configure section Go to the Configure section at the bottom left and add languages, we support over 105 languages as mentioned over here.

Don't forget to press the save button before switching the tab.

  1. Filter option for language for all available nodes

  1. Enable auto-detect language under tools, the bot will detect users language and automatically change to the respective language.

  1. Using the 'set language' node, create change language journey

Ask user about their preferred language via the quick reply node, can add languages as the buttons.

For every button under the button settings, add the ISO codes for the various languges.

Pass language from the quick replies to the set language node from the earlier quick reply node.

  1. Adding FAQs in other languages Can select the language filter available in FAQs section on the top right. Or bulk manage also allows FAQ answers upload in multiple languages.

  2. Training the bot in other languages It is not necessary but you can train your bot in multiple languages.
